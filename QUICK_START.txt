================================================================================
  IP ENRICHMENT FRAMEWORK - QUICK START
================================================================================

STEP 1: Add Your Shodan API Key
--------------------------------
Edit .env file and add:
SHODAN_API_KEY=your_shodan_key_here

STEP 2: Test the Setup
-----------------------
ti-platform enrichment-status

STEP 3: Enrich Your First IP
-----------------------------
ti-platform enrich-ip --ip 8.8.8.8

STEP 4: Enrich Multiple IPs
----------------------------
Create ips.txt with one IP per line, then:
ti-platform enrich-ips --file ips.txt --output results.json

================================================================================
COMMON COMMANDS
================================================================================

# Check status
ti-platform enrichment-status

# Enrich single IP
ti-platform enrich-ip --ip 1.2.3.4

# Save to file
ti-platform enrich-ip --ip 1.2.3.4 --output result.json

# Enrich batch
ti-platform enrich-ips --file ips.txt --output enriched.json

# Use specific modules only
ti-platform enrich-ip --ip 1.2.3.4 --modules geolocation,threat

# Increase parallel workers
ti-platform enrich-ips --file ips.txt --output results.json --parallel 20

# Export to CSV
ti-platform enrich-ips --file ips.txt --output results.csv
